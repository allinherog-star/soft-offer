# 快速评估功能使用指南

## 功能概述

快速评估功能允许您使用 AI（DeepSeek）快速生成项目的需求清单，包括系统模块、功能菜单、复杂度评估、优先级和详细说明。

## 使用步骤

### 1. 准备项目信息

在使用快速评估之前，请先在顶部导航栏填写：
- **项目名称**：您的项目名称
- **项目描述**：简要描述项目的目的和功能
- **行业应用**：选择适合的行业类型
- **交付端**：勾选需要开发的平台（PC端、Web端、H5、Android端、IOS端、小程序、后台）

### 2. 点击快速评估按钮

在左侧功能树面板顶部，点击"快速评估"按钮（带有闪电图标⚡）。

### 3. 自动复制提示词

对话框打开后，系统会：
- 自动生成针对您项目的提示词
- 自动复制提示词到剪贴板
- 显示3秒倒计时

### 4. 跳转到 DeepSeek

倒计时结束后，系统会：
- 自动在新标签页打开 DeepSeek 网站 (https://chat.deepseek.com/)
- 切换到输入阶段

### 5. 在 DeepSeek 中生成结果

在 DeepSeek 网站：
1. 粘贴刚才复制的提示词（Ctrl+V 或 Cmd+V）
2. 等待 AI 生成结果
3. 复制 AI 返回的 JSON 格式内容

### 6. 导入生成结果

返回本系统：
1. 在对话框的文本框中粘贴 DeepSeek 生成的 JSON 内容
2. 点击"导入"按钮
3. 系统会自动解析并添加所有需求模块到功能树中

## 提示词格式说明

系统生成的提示词会包含以下信息：
- 项目基本信息（名称、描述、行业、交付端）
- JSON 输出格式要求
- 示例结构

## DeepSeek 返回格式

DeepSeek 应该返回类似以下格式的 JSON：

```json
[
  {
    "name": "用户管理",
    "complexity": "中",
    "priority": "高",
    "isImportant": true,
    "remark": "用户管理模块包含用户的注册、登录、权限管理等核心功能",
    "children": [
      {
        "name": "用户注册",
        "complexity": "低",
        "priority": "高",
        "isImportant": false,
        "remark": "支持手机号、邮箱注册，需要验证码验证"
      }
    ]
  }
]
```

## 字段说明

- **name**: 模块或功能名称
- **complexity**: 复杂度（低、中、高、很高）
- **priority**: 优先级（低、中、高、很高）
- **isImportant**: 是否为重点功能（true/false）
- **remark**: 详细说明
- **children**: 子模块或功能列表（可选）

## 注意事项

1. **项目信息完整性**：填写越详细的项目信息，AI 生成的结果越准确
2. **JSON 格式**：确保复制的内容是完整的 JSON 格式
3. **支持 Markdown 代码块**：系统可以自动识别并提取 ```json ``` 代码块中的内容
4. **重新打开 DeepSeek**：如果需要重新生成，可以点击"重新打开 DeepSeek"按钮
5. **手动编辑**：导入后，您仍然可以在功能树中手动编辑、添加或删除节点

## 快捷操作

- 如果不想等待倒计时，可以点击"跳过等待"按钮直接进入输入阶段
- 提示词在对话框中始终可见，可以随时点击"复制"按钮重新复制
- 如果对生成的结果不满意，可以关闭对话框后重新尝试

## 常见问题

### Q: DeepSeek 没有返回 JSON 格式？
A: 请在提示词中明确要求返回 JSON 格式，或者在对话中追问"请以 JSON 格式输出"。

### Q: 导入时提示解析失败？
A: 请检查：
- 复制的内容是否完整
- JSON 格式是否正确（可以用在线 JSON 验证工具检查）
- 是否包含了多余的说明文字（系统会尝试自动提取 JSON 部分）

### Q: 可以多次使用快速评估吗？
A: 可以！每次导入都会将新的模块追加到现有的功能树中，不会覆盖已有内容。

### Q: 如何避免重复？
A: 建议在使用快速评估前先确认功能树是否已有相关模块，或者导入后手动删除重复的模块。
