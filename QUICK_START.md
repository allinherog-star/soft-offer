## 🚀 使用步骤

### 1️⃣ 启动项目
```bash
npm run dev
```
访问: http://localhost:3000

### 2️⃣ 填写项目信息
在顶部填写:
- **项目名称**: 例如 "电商平台"
- **项目描述**: 例如 "包含商品管理、订单处理..."

### 3️⃣ 点击 AI 生成
点击左侧 🪄 **AI 生成** 按钮

### 4️⃣ 自动操作
- ✅ 提示词自动复制到剪贴板
- ✅ 显示成功提示
- ✅ 自动跳转 DeepSeek 网站

### 5️⃣ 在 DeepSeek 使用
1. 在聊天框按 `Ctrl+V` (或 `Cmd+V`) 粘贴
2. 点击发送
3. 等待 AI 生成需求列表

### 6️⃣ 导入需求
复制生成的需求,返回系统手动创建
