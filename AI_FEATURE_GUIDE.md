# AI 智能生成需求功能使用指南

## 功能概述

本项目新增了 AI 智能生成需求模块功能,通过一键跳转到 DeepSeek 网站并自动复制提示词,让 AI 根据项目名称和描述自动生成完整的需求模块和功能列表。

**特点**:
- ✅ 无需配置 API Key
- ✅ 自动复制提示词到剪贴板
- ✅ 一键跳转到 DeepSeek 网站
- ✅ 简单易用,快速上手

## 使用步骤

### 1. 填写项目信息

在页面顶部导航栏填写:
- **项目名称**: 例如 "智能客服系统"
- **项目描述**: 例如 "一个支持多渠道接入、智能问答、工单管理的客服系统"

### 2. 点击 AI 生成按钮

1. **找到魔法按钮**: 在左侧功能树面板顶部,你会看到两个按钮:
   - 🔵 **添加需求** (主按钮,深蓝色)
   - 🪄 **AI 生成** (魔法按钮,紫色边框)

2. **点击 AI 生成**: 点击魔法按钮

3. **自动操作**:
   - ✅ 系统自动根据项目信息生成提示词
   - ✅ 提示词自动复制到剪贴板
   - ✅ 显示成功提示通知
   - ✅ 自动打开 DeepSeek 网站(新标签页)

### 3. 在 DeepSeek 粘贴提示词

1. DeepSeek 网站打开后,在聊天输入框粘贴提示词 (Ctrl+V / Cmd+V)
2. 点击发送
3. 等待 AI 生成需求模块列表

### 4. 复制生成结果并手动导入

1. 复制 DeepSeek 生成的需求列表
2. 返回本系统
3. 根据生成的结果,手动创建需求模块和功能

## 使用示例

- **项目名称**: 电商平台

项目名称: 电商平台
项目描述: 面向C端用户的在线购物平台,包含商品浏览、购物车、订单管理、支付、物流跟踪等功能

请按照以下格式生成需求模块和功能列表:

1. 模块名称
1.1 子功能1

2. 模块名称
2.1 子功能1

## 注意事项

1. **浏览器兼容性**: 需要现代浏览器支持剪贴板 API
2. **项目信息**: 项目描述越详细,生成的需求越准确
3. **手动导入**: 目前需要手动将生成的需求录入系统
4. **网络要求**: 需要能访问 DeepSeek 网站 (https://chat.deepseek.com/)

## 常见问题

### Q: 为什么需要跳转到 DeepSeek?
A: 这样设计的优势:
- 无需配置 API Key,更简单
- 可以实时看到 AI 生成过程
- 可以和 AI 互动,调整需求
- 完全免费使用

### Q: 如果复制失败怎么办?
A: 会显示错误提示,可以手动选择并复制提示词

### Q: 是否支持其他 AI?
A: 提示词是通用的,可以复制到任何 AI 聊天工具使用 (ChatGPT、Claude 等)

### Q: 未来会自动导入吗?
A: 计划在后续版本中增加自动解析和导入功能

## 技术实现

- **前端**: React + Toast 通知组件
- **剪贴板**: Navigator Clipboard API
- **跳转**: window.open() 新标签页打开
- **提示词**: 模板字符串动态生成

## 优势

1. **零配置**: 无需 API Key,开箱即用
2. **可交互**: 可以在 DeepSeek 与 AI 对话调整需求
3. **灵活性**: 生成的内容可以人工审核后再导入
4. **免费**: 完全免费使用,无额外成本
