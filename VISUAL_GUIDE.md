## Toast 通知效果

成功复制时显示:

```
┌──────────────────────────────┐
│  ✅ 提示词已复制              │
│  即将跳转到 DeepSeek,         │
│  请在聊天框粘贴提示词         │
└──────────────────────────────┘
      ↑
   淡入淡出动画
   3秒后自动消失
```

失败时显示:

```
┌──────────────────────────────┐
│  ❌ 复制失败                  │
│  请手动复制提示词             │
└──────────────────────────────┘
```

## 颜色方案

## 技术细节

### 剪贴板 API
```javascript
await navigator.clipboard.writeText(prompt);
```

### 窗口跳转
```javascript
window.open('https://chat.deepseek.com/', '_blank');
```

### Toast 通知
```javascript
toast({
  title: "✅ 提示词已复制",
  description: "即将跳转到 DeepSeek，请在聊天框粘贴提示词",
  duration: 3000,
});
```
